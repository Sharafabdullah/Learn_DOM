Introduction to the HTML DOM

The HTML DOM (Document Object Model) is the browser's programming interface for HTML. When a web page is loaded, the browser creates a DOM of the page, which is an object-oriented representation of the web page that can be modified with a scripting language like JavaScript.

What is the DOM?

The DOM represents the document as a logical tree of nodes. Each node is an object representing a part of the document, like an element, an attribute, or a piece of text. This structured representation allows you to interact with the page dynamically and create powerful web applications.

Why is it Powerful?

  • Change all HTML elements in the page
  • Change all HTML attributes in the page
  • Change all CSS styles in the page
  • Remove or add HTML elements and attributes
  • React to all existing HTML events
  • Create new HTML events

DOM Methods & Properties

HTML DOM methods are actions you can perform on HTML elements, while DOM properties are values of HTML elements that you can get or set. Together, they form the programming interface that allows you to dynamically interact with web pages using JavaScript.

What are Methods?

Methods are actions you can perform on HTML elements. Think of them as functions that do something - like finding an element, adding content, or removing elements from the page. Methods are called with parentheses and can accept parameters to customize their behavior.

What are Properties?

Properties are values that you can get or set. They represent the current state of an HTML element - like its content, style, or attributes. Properties don't use parentheses and can be read or modified to change how elements appear and behave.

Programming Interface

In the DOM, all HTML elements are defined as objects. Each object has properties (data) and methods (functions) that create the programming interface. This object-oriented approach makes it easy to manipulate web pages programmatically.

Common Examples

  • getElementById() - Find element by ID
  • innerHTML - Get/set element content
  • createElement() - Create new elements
  • appendChild() - Add elements to page
  • style - Modify CSS properties
  • className - Change CSS classes

The HTML DOM Document Object

The document object is the root of the HTML DOM and the owner of all other objects in your web page. If you want to access or manipulate any element in an HTML page, you always start with the document object. It represents your web page and provides methods and properties to find, change, add, or remove elements, as well as to handle events and interact with the browser.

Finding HTML Elements

Method Description
document.getElementById(id) Find an element by element id
document.getElementsByTagName(name) Find elements by tag name
document.getElementsByClassName(name) Find elements by class name

Finding and Accessing DOM Elements

To manipulate HTML with JavaScript, you first need to find the elements you want to work with. The DOM provides several powerful methods to access elements by id, tag name, class name, CSS selectors, or object collections.

Common ways to find elements:

By ID

Find a single element by its unique id attribute.

const el = document.getElementById("intro");

By Tag Name

Find all elements with a given tag name.

const ps = document.getElementsByTagName("p");

By Class Name

Find all elements with a specific class.

const items = document.getElementsByClassName("intro");

By CSS Selector

Find all elements matching a CSS selector.

const items = document.querySelectorAll("p.intro");

By Object Collections

Access elements via built-in collections like forms, images, etc.

const form = document.forms["frm1"];

More Useful Info

  • getElementById returns a single element or null if not found.
  • getElementsByTagName, getElementsByClassName, and querySelectorAll return collections (array-like, but not arrays).
  • querySelector returns only the first matching element.
  • You can loop through collections with for...of or forEach (after converting to an array).
  • Object collections include document.forms, document.images, document.links, etc.
  • See MDN: Document API for more.

Changing HTML Content

The HTML DOM allows JavaScript to change the content of HTML elements. The easiest way to modify the content of an HTML element is by using the innerHTML property.

Changing HTML Content

To change the content of an HTML element, use this syntax:
