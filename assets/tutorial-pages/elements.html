Finding and Accessing DOM Elements

To manipulate HTML with JavaScript, you first need to find the elements you want to work with. The DOM provides several powerful methods to access elements by id, tag name, class name, CSS selectors, or object collections.

Common ways to find elements:

By ID

Find a single element by its unique id attribute.

const el = document.getElementById("intro");

By Tag Name

Find all elements with a given tag name.

const ps = document.getElementsByTagName("p");

By Class Name

Find all elements with a specific class.

const items = document.getElementsByClassName("intro");
