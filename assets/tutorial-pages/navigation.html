DOM Navigation

With the HTML DOM, you can navigate the node tree using node relationships. Understanding how to traverse and navigate between DOM nodes is essential for dynamic web development.

DOM Nodes

According to the W3C HTML DOM standard, everything in an HTML document is a node:

  • The entire document is a document node
  • Every HTML element is an element node
  • The text inside HTML elements are text nodes
  • Every HTML attribute is an attribute node (deprecated)
  • All comments are comment nodes

Node Relationships

The nodes in the node tree have a hierarchical relationship to each other. The terms parent, child, and sibling are used to describe the relationships.

  • The top node is called the root
  • Every node has exactly one parent (except root)
  • A node can have multiple children
  • Siblings are nodes with the same parent

Navigation Properties

You can use the following node properties to navigate between nodes with JavaScript:

  • parentNode - Access parent element
  • childNodes[number] - Access child by index
  • firstChild - First child node
  • lastChild - Last child node
  • nextSibling - Next sibling node
  • previousSibling - Previous sibling node

Node Types

The most important nodeType properties are:

Node Type Example
ELEMENT_NODE 1 <h1>Title</h1>
ATTRIBUTE_NODE 2 class="heading" (deprecated)
TEXT_NODE 3 Text content
COMMENT_NODE 8 <!-- comment -->
DOCUMENT_NODE 9 The HTML document itself
DOCUMENT_TYPE_NODE 10 <!DOCTYPE html>
