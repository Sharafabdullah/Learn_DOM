HTML DOM CSS Styling

The HTML DOM allows JavaScript to change the style of HTML elements. JavaScript can modify CSS properties directly, enabling dynamic styling and interactive visual effects.

Changing HTML Style

To change the style of an HTML element, use this syntax:

document.getElementById(id).style.property = new style

Using Events

The HTML DOM allows you to execute code when an event occurs. Events are generated by the browser when "things happen" to HTML elements:

  • An element is clicked on
  • The page has loaded
  • Input fields are changed

You will learn more about events in the next chapter of this tutorial.

Common CSS Properties

Property Description
color Sets the text color
backgroundColor Sets the background color
fontSize Sets the font size
display Controls element visibility and layout
visibility Controls element visibility without affecting layout
border Sets the border style

Important Notes

  • CSS property names in JavaScript use camelCase (e.g., backgroundColor instead of background-color)
  • Style changes are applied immediately when the code executes
  • You can read current style values using getComputedStyle()
  • Style changes through JavaScript take priority over CSS stylesheets
