Introduction to the HTML DOM

The HTML DOM (Document Object Model) is the browser's programming interface for HTML. When a web page is loaded, the browser creates a DOM of the page, which is an object-oriented representation of the web page that can be modified with a scripting language like JavaScript.

What is the DOM?

The DOM represents the document as a logical tree of nodes. Each node is an object representing a part of the document, like an element, an attribute, or a piece of text. This structured representation allows you to interact with the page dynamically and create powerful web applications.

Why is it Powerful?

  • Change all HTML elements in the page
  • Change all HTML attributes in the page
  • Change all CSS styles in the page
  • Remove or add HTML elements and attributes
  • React to all existing HTML events
  • Create new HTML events
