HTMLCollection

The HTMLCollection object is an array-like list of HTML elements returned by methods like getElementsByTagName(). Understanding how to work with HTMLCollections is essential for manipulating multiple DOM elements efficiently.

HTMLCollection Properties

HTMLCollection objects provide several key features:

  • Array-like access with index numbers
  • length property for counting elements
  • Live collection - updates automatically
  • Access elements by index: collection[0]
  • Can be iterated with for loops

Important Differences

HTMLCollection vs Array - key distinctions:

  • NOT a true JavaScript array
  • No array methods (push, pop, join)
  • Index access works: collection[0]
  • Has length property
  • Can use for loops

HTMLCollection Methods & Properties

Available properties and methods:

Property/Method Description Example
length Number of elements collection.length
[index] Access by index collection[0]
item(index) Access by index (method) collection.item(0)
namedItem(name) Access by name/id collection.namedItem("myId")

Best Practices

  • Always check length before accessing elements
  • Use for loops for simple iteration
  • Convert to array for complex operations
  • Remember collections are live and update automatically
  • Cache length in loops for better performance
  • Use modern methods like querySelectorAll() for static collections
