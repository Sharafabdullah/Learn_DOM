DOM Methods & Properties

HTML DOM methods are actions you can perform on HTML elements, while DOM properties are values of HTML elements that you can get or set. Together, they form the programming interface that allows you to dynamically interact with web pages using JavaScript.

What are Methods?

Methods are actions you can perform on HTML elements. Think of them as functions that do something - like finding an element, adding content, or removing elements from the page. Methods are called with parentheses and can accept parameters to customize their behavior.

What are Properties?

Properties are values that you can get or set. They represent the current state of an HTML element - like its content, style, or attributes. Properties don't use parentheses and can be read or modified to change how elements appear and behave.

Programming Interface

In the DOM, all HTML elements are defined as objects. Each object has properties (data) and methods (functions) that create the programming interface. This object-oriented approach makes it easy to manipulate web pages programmatically.

Common Examples

  • getElementById() - Find element by ID
  • innerHTML - Get/set element content
  • createElement() - Create new elements
  • appendChild() - Add elements to page
  • style - Modify CSS properties
  • className - Change CSS classes
